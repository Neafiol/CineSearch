# CineSearch
Приложение для поиска кино на вечер в несколько кликов. Рабочая версия алгоритма в интерфесе telegram бота по ссылке t.me/cinesearchbot

### Технологии
+ python3
+ Telegram Bot API
+ Postgres SQL
+ Peewee ORM
+ Selenium
+ asincio
+ Sanic web fraemwork
+ Владение REST API
+ pandas
+ numpy
+ matplotlib
+ CatBoostClassifier
+ Xgboost
+ sklearn.AffinityPropagation
+ sklearn.KNN
